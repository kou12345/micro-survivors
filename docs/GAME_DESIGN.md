# Micro Survivors - ゲームデザインドキュメント

## 基本情報

| 項目 | 内容 |
|------|------|
| タイトル | Micro Survivors（仮） |
| ジャンル | ローグライト・サバイバーアクション |
| プラットフォーム | ブラウザ（Cloudflare Workers/Pages） |
| 1プレイ時間 | 最大15分 |
| 技術 | HTML5 Canvas / JavaScript / ピクセルアート |

## コンセプト

ミクロの世界で免疫細胞となり、押し寄せるウイルスやバクテリアから生き延びる。自動攻撃×移動操作のシンプルな操作で、大量の敵をなぎ倒す爽快感を楽しむ。

## プレイヤーキャラクター

| キャラ | タイプ | 固有武器 | 特徴 |
|--------|--------|----------|------|
| 白血球くん | バランス型 | 貪食パンチ（近距離） | 初心者向け |
| T細胞 | 攻撃特化 | 抗体ミサイル（誘導弾） | 高火力・低耐久 |
| マクロファージ | タンク型 | 巨大化踏み潰し | 遅いが硬い |
| バクテリオファージ | トリッキー | 敵に取り憑き爆発 | 上級者向け |
| アメーバ | サポート型 | 分裂して囮生成 | 生存特化 |

※プロトタイプでは**白血球くん**のみ実装

## 武器システム

| 武器 | 攻撃パターン | 進化後 |
|------|--------------|--------|
| 抗体 | プレイヤー周囲を回転 | 抗体チェーン |
| 酵素弾 | 前方に発射 | 連射酵素砲 |
| ATP爆弾 | 設置→時間差爆発 | メガATP |
| 繊毛ウィップ | 鞭状に薙ぎ払い | 超振動繊毛 |
| 粘液トラップ | 床設置、敵減速 | 毒性粘液 |

## パッシブアイテム

| アイテム | 効果 |
|----------|------|
| ミトコンドリア | 攻撃力アップ |
| リボソーム | 経験値アップ |
| 細胞膜強化 | 防御力アップ |
| 鞭毛 | 移動速度アップ |
| 核膜 | クールダウン短縮 |

## レベルアップシステム

1. 敵を倒す → 経験値（光る粒子）ドロップ
2. 経験値を集める → レベルアップ
3. 3つの選択肢から武器/パッシブを選ぶ
4. 同じ武器を選ぶと強化（レベル最大8）

## ビジュアルスタイル

- **ピクセルアート**
- 背景：淡い緑/青グラデーション、浮遊する泡
- 敵：カラフルでポップ（怖くない）
- エフェクト：キラキラ粒子、ポップなダメージ数字
- UI：丸みのあるデザイン、顕微鏡の覗き窓風フレーム
